## 7. 핵심 파라미터 빠른참조 (제출 시 그대로 입력)

| 항목 | 값 |
|---|---|
| 토큰명 / 티커 | **BEAN** |
| 총 발행량 | 1,000,000,000 (10억, 고정) |
| 소수점 | 7 (1 BEAN = 10,000,000 units, i128) |
| 세일가격 | 0.01 Pi / BEAN (`1 Pi = 100 BEAN`) |
| 목표 조달 | 소프트캡 2,000,000 Pi / 하드캡 4,000,000 Pi |
| 약정 한도 | 최소 1 Pi / 최대 500 Pi (Fair-Access) |
| 분배 | 세일 40% · 리저브 25% · 유동성 15% · 마케팅 12% · 팀 8% |
| 팀 베스팅 | 6M cliff + 36M 선형 |
| 유동성 페어 | **BEAN/Pi 단독** (Pi 외 자산 페어 없음) |
| 발행 주체 | 개인 (아나킨) |
| 체인/컨트랙트 | Pi Mainnet (Stellar) / Soroban |
