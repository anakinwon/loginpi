# Youth Protection Policy

**버전**: v1.0  
**시행일**: 2026-06-12  
**작성일**: 2026-06-12

## 제1장 총칙

### 제1조 목적
본 정책은 Cafe.pi(이하 "서비스")가 청소년보호법, 정보통신망법, 아동복지법 등을 준수하여 만 18세 미만의 미성년자(이하 "청소년")의 건강한 이용 환경을 보장하고 유해 콘텐츠로부터 보호하기 위해 수립됩니다.

### 제2조 용어 정의
1. **청소년**: 만 14세 이상 만 18세 미만의 사람
2. **어린이**: 만 14세 미만의 사람
3. **유해 콘텐츠**: 청소년에게 해로운 음란, 폭력, 도박, 약물 관련 정보
4. **법정대리인**: 부모, 보호자 등 청소년의 상법상 대리인

### 제3조 적용 범위
본 정책은 다음에 적용됩니다:
- 청소년 회원 가입 및 계정 관리
- 청소년이 접근 가능한 콘텐츠 필터링
- 청소년 개인정보 수집 및 보호
- 청소년 결제 기능 제한
- 부모/보호자 동의 절차

## 제2장 청소년 회원 관리

### 제4조 청소년 회원가입 정책

#### 4-1. 연령 제한
- **어린이(만 14세 미만)**: 서비스 가입 **불가**
  - 가입 시도 시 "이 서비스는 만 14세 이상만 이용 가능합니다" 메시지 표시
  - 가입 페이지에서 "생년월일" 입력 필드가 자동 차단
  - 생년월일 변조로 가입한 어린이 적발 시 즉시 계정 삭제

- **청소년(만 14~17세)**: 법정대리인 동의 필수

#### 4-2. 청소년 가입 절차
1. 생년월일 입력 (만 14~17세 인정)
2. "부모/보호자 동의 필요" 안내 표시
3. 법정대리인 이메일 입력
4. 회사가 법정대리인에게 동의 이메일 발송
5. 법정대리인이 이메일 내 "동의" 링크 클릭
6. 청소년 계정 활성화

#### 4-3. 법정대리인 동의 항목
동의서에 다음 내용을 포함합니다:
- 개인정보(생년월일, 이메일) 수집 및 이용
- 서비스 이용약관 동의
- 개인정보처리방침 동의
- **특별 조항**: "청소년이 유해 콘텐츠 차단된 상태로 이용하며, 결제 기능이 제한됨을 인지합니다"

### 제5조 청소년 계정 기능 제한

#### 5-1. 채팅 기능
| 기능 | 청소년 제한 | 비고 |
|---|---|---|
| 1:N 카페 채팅 | **제한 없음** | 커뮤니티 참여 장려 |
| 1:1 개인 채팅 | **차단** | 대면 접촉 위험 방지 |
| 보이스채팅(1:N) | **제한 없음** | 성별/나이 공개 필수 |
| 보이스채팅(1:1) | **차단** | 사생활 침해 방지 |

#### 5-2. 게시판 기능
| 게시판 | 청소년 제한 | 비고 |
|---|---|---|
| 공지/공략 | **제한 없음** | 읽기만 가능 |
| 자유글 | **제한** | 글 작성 가능, 댓글 시 본명 또는 보호자명 표시 필수 |
| 장터 | **제한** | 상품 구매 가능, 판매 불가 (보호자 승인 시만 판매 가능) |

#### 5-3. 마켓플레이스
| 기능 | 청소년 제한 | 비고 |
|---|---|---|
| 상품 **구매** | **제한 없음** | 부모/보호자 결제 승인 필수 |
| 상품 **판매** | **차단** (기본) | 보호자 신청 시 판매 허용 |
| **결제 한도** | **월 50,000 Pi** | 보호자가 증감 설정 가능 |

#### 5-4. 설정 예시
청소년 계정 마이페이지 > 보호자 설정:
```
[ 보호자 이메일 ] : parent@example.com
[ 월 결제 한도 ] : 50,000 Pi (변경 가능)
[ 음란물 자동 차단 ] : ON
[ 폭력/협박 콘텐츠 필터 ] : ON
[ 게임/도박 관련 콘텐츠 필터 ] : ON
```

### 제6조 보호자 접근 및 모니터링

#### 6-1. 보호자 포털
법정대리인은 자녀 계정 모니터링 포털("MyChild" 기능)에 접근할 수 있습니다:

**포털 주요 기능**:
1. **활동 로그**: 자녀의 일일 로그인, 채팅방 방문, 게시물 작성 기록
2. **친구 목록**: 자녀가 채팅하는 사람 목록 (익명 처리)
3. **결제 관리**: 월 결제 한도 설정, 결제 내역 확인
4. **콘텐츠 필터**: 차단할 콘텐츠 유형 선택
5. **알림 설정**: 의심 활동 시 보호자 이메일 알림

#### 6-2. 의심 활동 알림
보호자에게 자동 알림되는 경우:
- 자녀가 비정상 접속 시간대(새벽 2시~5시) 활동
- 성인용 콘텐츠 접근 시도
- 개인정보(주소, 전화) 공유
- 결제 한도 초과 시도
- 타인에게 협박/욕설 신고

#### 6-3. 보호자 개입 권한
보호자는 다음 조치를 실시할 수 있습니다:
1. **강제 로그아웃**: 자녀 계정 즉시 로그아웃
2. **기능 일시 제한**: 채팅, 결제 기능 임시 차단
3. **휴계정 전환**: 계정 일시 정지 (자녀 재활성화 불가)
4. **계정 삭제**: 보호자가 자녀 계정 삭제 신청 (법적 효력 확인)

## 제3장 유해 콘텐츠 필터링 및 차단

### 제7조 자동 필터링 시스템

#### 7-1. 필터링 범위
청소년 계정에 다음 콘텐츠는 **자동으로 차단**되거나 **흐리게** 표시됩니다:

| 콘텐츠 유형 | 필터링 방식 | 노출 시 | 비고 |
|---|---|---|---|
| **음란물** | 완전 차단 (게시물 숨김) | 경고 + 접근 불가 | AI 자동 탐지 + 수동 검증 |
| **폭력 영상** | 흐리게 처리 + 경고 | 클릭 시 "성인 콘텐츠" 알림 | 사용자 선택 가능 |
| **협박/괴롭힘** | 완전 차단 (신고 처리) | 신고 | 즉시 삭제 + 발신자 제재 |
| **약물/도박** | 완전 차단 (정보 제공 금지) | 접근 불가 | 교육 목적 콘텐츠는 예외 검토 |
| **자해/자살 조장** | 완전 차단 + 정신건강 리소스 제공 | 대신 상담 전화 안내 | 위기상담 전화 제시 |

#### 7-2. 필터링 예외 (허용 콘텐츠)
다음은 청소년도 접근 가능합니다:
- 교육 목적 정보 (성교육, 건강 정보, 법률 상담)
- 뉴스/언론 기사 (객관적 보도)
- 예술 작품 (영화, 드라마 정보 — 19금 제외)
- 사회 문제 토론 (폭력/혐오 표현 제외)

#### 7-3. 필터 오탐 신고
- 청소년 또는 보호자가 정당한 콘텐츠가 잘못 차단되었다고 판단하면 신고 가능
- 신고 후 3일 이내 인적 검토 후 복구 또는 유지 결정

#### 8-1. 자체 등급 시스템
게시물·채팅에 다음 라벨 표시:

| 등급 | 표시 | 청소년 차단 | 비고 |
|---|---|---|---|
| **전체 이용가** | 🟢 ALL | 차단 없음 | 기본값 |
| **12세 이상** | 🟡 12+ | 차단 없음 | 가벼운 폭력/욕설 |
| **15세 이상** | 🟠 15+ | 차단 (보호자 승인 시 해제 가능) | 중대한 폭력/성인 암시 |
| **18세 이상** | 🔴 18+ | 차단 (청소년 절대 접근 불가) | 음란물, 심각한 폭력 |

- **자동 라벨**: AI 기반 언어 분석 (욕설, 폭력어 감지)
- **수동 라벨**: 신고 후 직원이 수동 라벨링
- **광고주 선택**: 게시자가 자신의 콘텐츠 등급 제안 가능 (회사 검증 후 확정)

## 제4장 청소년 개인정보 보호

### 제9조 최소 수집 원칙

청소년으로부터 수집하는 개인정보:

| 정보 | 수집 여부 | 근거 | 보유 기간 |
|---|---|---|---|
| **생년월일** | ✓ (필수) | 연령 확인 | 계정 유지 기간 + 3년 |
| **법정대리인 이메일** | ✓ (필수) | 동의, 모니터링 포털 | 계정 유지 기간 + 1년 |
| **유전자 정보** | ✗ (수집 금지) | 개인정보보호법 | - |
| **재정 정보** (신용카드) | ✗ (간접 수집만) | 부모 결제 통해서만 | 거래 기록 5년 |
| **위치 정보** | ⚠️ (선택) | 별도 동의 필수 | 1년 |

### 제10조 개인정보 공개 금지

청소년 회원의 다음 정보는 **절대 공개되지 않습니다**:
- 생년월일 (나이 범위 "청소년" 표시만)
- 법정대리인 연락처
- 실명 (닉네임만 표시)
- 학교/주소/전화번호 (자신이 공개하지 않는 한)
- 결제 정보 (거래 기록만 보호자가 열람 가능)

### 제11조 청소년 데이터 삭제

#### 11-1. 계정 삭제 시 처리
청소년이 계정을 삭제하면:
1. 개인정보 즉시 논리삭제 (del_yn = 'Y')
2. 닉네임은 "[삭제된 사용자]"로 처리
3. 작성한 게시물/채팅은 **콘텐츠만 유지**, 작성자 정보 삭제
4. 거래 기록은 법정 보존 기간(5년) 유지

#### 11-2. 만 18세 전환 처리
청소년이 만 18세가 되면:
1. 자동 알림 발송 (성인 계정 전환 안내)
2. 기능 제약 해제 옵션 제공
3. 법정대리인 모니터링 포털 폐기 (자녀 동의 시에만)

## 제5장 결제 및 구독 관리

### 제12조 청소년 결제 보호

#### 12-1. 부모 승인 필수
청소년의 모든 결제:
1. 결제 전 **부모/보호자 승인 요청** 화면 표시
2. 보호자 이메일로 승인 링크 발송
3. 보호자가 24시간 내 승인/거부 선택
4. 승인 후에만 결제 진행

**예시 메시지**:
```
[결제 승인 요청]

청소년 회원: park_john (14세)
상품: 구독 서비스 "카페 VIP" (월 10,000 Pi)
요청 시간: 2026-06-12 14:30

[승인] [거부] [결제 한도 변경]
```

#### 12-2. 월 결제 한도
- **기본값**: 월 50,000 Pi
- **최대값**: 월 100,000 Pi
- **최소값**: 월 0 Pi (차단)
- **변경 권한**: 부모/보호자만 가능
- **초과 시도**: 차단 + 부모 알림

#### 12-3. 결제 내역 투명성
보호자는 자녀의 다음을 확인 가능:
- 월별 총 결제액
- 상품별 결제 내역 (날짜, 금액, 상품명)
- 구독 현황 (활성/해지)
- 환불 기록

---

### 제13조 구독 자동 갱신 정책

#### 13-1. 청소년 구독의 특수성
- 구독 시 **자동 갱신 기본값: OFF** (성인은 ON)
- 청소년이 자동 갱신을 원하면 부모 승인 필수
- 월 1회 갱신일 3일 전 부모와 청소년 모두에게 상기 알림

#### 13-2. 구독 취소
청소년이 구독을 취소하면:
1. 즉시 갱신 중단
2. 부모에게 취소 알림
3. 남은 기간 일할 환불 자동 처리

## 제6장 안전 및 신고 체계

### 제14조 청소년 신고 채널

"신고" 버튼:

#### 14-2. 보호자 신고

- 의심 사용자

- 성인의 접근 시도
- 신고 후 3일 이내 조사 완료 및 알림

#### 14-3. 공식 신고 채널
- 이메일: anakin.won@gmail.com (제목: "[청소년 보호] 신고")
- 전화: (번호 미정)

### 제15조 위험 상황 대응

#### 15-1. 성인의 청소년 대상 접근 시도
다음 행위 적발 시 **즉시 적발자 계정 차단 + 경찰 신고**:
- 나이 속여서 1:1 대화 유도
- 성적 착취 목적 개인정보 요청
- 만남 제의
- 성적 콘텐츠 공유

#### 15-2. 청소년의 위험 행위
청소년이 다음 행위:
- **자해/자살 관련 게시**: 즉시 게시물 삭제 + 정신건강 상담 리소스 제공
  - 정신건강위기상담전화: 1393 (정신건강 중기관)
  - 생명의전화: 1393
  
- **약물 관련 언급**: 게시물 차단 + 약물 위험성 교육 자료 제공
  
- **괴롭힘 피해 신고**: 즉시 가해자 추적 + 경찰 신고 + 보호 조치

#### 15-3. 응급 상황
- 청소년이 "지금 자살하겠다" 등 긴급 상황 게시 시 즉시 경찰 신고
- 게시물 삭제 아님 (증거 보존)

## 제7장 교육 및 인식 제고

### 제16조 청소년 디지털 리터러시 교육

#### 16-1. 앱 내 팁 제공
신규 청소년 회원 가입 후:
1. 온보딩 화면: "안전한 인터넷 사용 가이드" (5분)
2. 월간 안전 팁: 마이페이지 > "청소년 보호" 섹션
3. 상황별 안내:
   - "낯선 사람이 개인정보를 요청했어요" → 신고 방법
   - "친구가 내 몰래 사진을 올렸어요" → 삭제 신청 방법
   - "누군가 협박하고 있어요" → 차단 및 신고 방법

#### 16-2. 부모/보호자 교육 자료
법정대리인에게 제공:
- "자녀 온라인 활동 모니터링 가이드"
- "청소년 사이버폭력 대응 방법"
- "디지털 금융 교육" (가상자산 이해)
- PDF 다운로드 + 유튜브 튜토리얼

## 제8장 정부 신고 및 협력

### 제18조 정부 기관 신고

회사는 다음 상황에서 즉시 정부 기관에 신고합니다:

| 상황 | 신고 기관 | 기한 |
|---|---|---|
| **아동 성착취 자료** | 경찰청 + 아동학대 신고 전화 1391 | 즉시 |
| **청소년 사이버 성폭력** | 경찰청 + 한국정보통신진흥협회 | 24시간 내 |
| **협박/탈취** | 경찰청 | 24시간 내 |
| **신상 공개** | 경찰청 | 24시간 내 |
| **개인정보 유출** | 개인정보보호위원회 + 경찰청 | 5일 내 |

### 제19조 협력 기관

회사는 다음 기관과 협력합니다:
- **한국정보통신진흥협회**: 청소년보호 정책 수립 및 신고
- **경찰청 범죄분석과**: 사이버 범죄 대응
- **여성가족부**: 아동 학대·보호 정책 연계
- **한국정신건강위기상담센터**: 자해 위험 청소년 연계
