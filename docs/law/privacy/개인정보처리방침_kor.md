## 제6장 개인정보 보안

### 제8조 보안 조치
1. **암호화**: 
   - 통신 보안: HTTPS/TLS 1.3 이상
   - 저장 암호화: 민감한 정보(Pi 지갑 주소 등)는 AES-256 암호화
   
2. **접근 제어**:
   - 업무상 필요한 직원만 개인정보 접근 권한 부여
   - 정기적 권한 검토 및 회수

3. **침입 탐지**:
   - 악성코드 탐지 시스템
   - 실시간 로그 모니터링
   - 월 1회 보안 감시

4. **개인정보 유출 사고 대응**:
   - 사고 발생 즉시 내부 신고
   - 경찰청, 방송통신위원회 신고 (5일 이내)
   - 영향받은 회원에게 통보 (개인정보보호법 제34조)

### 제9조 정보주체의 보안 의무
회원은 다음을 준수해야 합니다:
- 강력한 비밀번호 설정 (영문, 숫자, 특수문자 조합)
- 타인과 비밀번호 공유 금지
- 공용 기기 사용 후 로그아웃 확인
- 의심스러운 로그인 시 즉시 신고
