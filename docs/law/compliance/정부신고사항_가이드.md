## 개요

Cafe.pi는 다음 특성을 보유합니다:
- **기술스택**: Next.js 16 + Supabase (싱가포르) + Pi Browser + Pi SDK
- **주요 기능**: 채팅(1:N 보이스채팅), 게시판(4종), Pi 결제(팁/구독/마켓플레이스)
- **서비스 형태**: 개인 운영 B2C 커뮤니티 + 마켓플레이스
- **결제 구조**: Pi SDK P2P 결제 중개 (회사 직접 수탁 없음 — **VASP 해당 가능성 낮음이나 추가 검토 필수**)

이 가이드는 각 신고 사항의 **해당 여부, 신고 기관, 필요 서류, 신고 기한**을 정리합니다.

### 3. VASP 신고 (조건부 — 전문 검토 필수) ⚠️

#### 3-1. VASP 해당 여부 판단

**VASP(Virtual Asset Service Provider)**: 특정금융정보법 제2조에 정의된 가상자산 서비스 제공자

**해당 판단 기준**:

| 조건 | Cafe.pi | 해당 여부 |
|---|---|---|
| **자산 수탁** | 회사가 Pi 직접 보관? | ❌ 아니오 (P2P) |
| **거래소** | Pi ↔ 원화 교환 제공? | ❌ 아니오 (P2P만) |
| **지갑 서비스** | 회사 지갑 제공? | ❌ 아니오 (사용자 지갑만) |
| **송금 중개** | 사용자 간 송금 중개? | ✅ 예 (팁, 구독료) |
| **마켓플레이스** | 상품 거래 시 Pi 수령? | ⚠️ 확인 필요 |

**예비 판단**:
- **가능성 낮음**: 회사가 Pi를 직접 수탁하지 않고 P2P 결제만 중개
- **가능성 있음**: 마켓플레이스에서 회사가 판매자의 Pi를 일시 보유하는 경우
- **결론**: **전문 법률 검토 필수**

#### 3-2. VASP 신고 요구 시 절차

만약 금융정보분석원(FIU)에서 VASP 신고 요구:

| 항목 | 내용 |
|---|---|
| **신고 기관** | 금융정보분석원(FIU) / 금융감독위원회 |
| **신고 기한** | 통지 후 **30일 이내** (특례: 90일 가능) |
| **필요 서류** | VASP 신고서, 사업자등록증, 대표자 신분증, AML/CFT 정책, 보안 감시 계획 |
| **수수료** | 없음 (별도 감시료 연 1,000만원 예상) |
| **연간 의무** | AML/CFT 컴플라이언스 보고서 제출 |

**비용 추정**:
- 신고 수수료: 0원
- 연간 감시료: 약 1,000만원 (거래액 기반)
- 컴플라이언스 인력: 전담자 1인 필요 (또는 외부 용역)

#### 3-3. 현재 권장 조치
1. **법률 검토**: 전자금융감독 전문 로펌에 VASP 해당성 검토 의뢰
   - 예상 비용: 200-500만원
   - 기간: 2-4주
   
2. **코드 분석**: `/api/payments/`, `/api/subscriptions/` 로직 검토
   - 회사가 중간에 Pi를 보유하는가?
   - 마켓플레이스 거래 흐름 재확인

3. **사전 신고 검토**: 신고 필요 시 FIU 사전 상담
   - 금융감독위원회 1332 (금감원 상담센터)
   - 공식 이메일: fiu@fiu.go.kr

### 4. 부가통신사업 신고 (조건부 — 면제 가능성 있음)

| 항목 | 내용 |
|---|---|
| **해당 여부** | ⚠️ **조건부** — 음성/영상 통신 제공 시 |
| **근거 법령** | 정보통신망법 제31조 |
| **신고 기관** | 과학기술정보통신부 (MOIS) |
| **필요 조건** | 1:N 또는 1:1 실시간 음성/영상 서비스 제공 |

**Cafe.pi 해당성**:
- **보이스채팅(1:N 공개 채팅)**: 일반 채팅 범주 → 신고 불필요
- **보이스채팅(1:1 개선)**: 실시간 통신 → **신고 검토 필요**
- **소규모 면제**: 월 가입자 1,000명 미만 + 거래액 연 1억원 미만 → 신고 면제

**현재 권장**:
- [ ] 월 활성 사용자 수 집계
- [ ] 음성 통신 기술 아키텍처 확인 (WebRTC 자체 인프라 vs. 제3자 서비스)
- [ ] 추정 거래액 계산

**신고 필요 시**:
| 항목 | 내용 |
|---|---|
| **신고 기한** | 사업 개시 **이전** |
| **필요 서류** | 부가통신사업신고서, 기술 명세서, 보안 정책 |
| **수수료** | 무료 (초기), 연 감시료 약 100-200만원 |

## 외부 법률 자문 가이드

### 추천 검토 사항

**1단계 (필수)**:
- VASP 신고 여부 판정 → **전자금융 전문 로펌** (2-4주, 200-500만원)
- Pi 결제 구조 법적 타당성 → **암호자산 전문가** (1-2주, 100-300만원)

**2단계 (권장)**:
- 약관 최종 검토 → **소비자보호 전문 로펌** (1주, 100-200만원)
- 청소년 보호 정책 검토 → **미디어/보호 전문 로펌** (1주, 100-200만원)

## FAQ

### Q1: Pi 결제는 반드시 VASP 신고가 필요한가?
**A**: 현재 구조(P2P 중개)로는 필요 없을 가능성이 높으나, 마켓플레이스에서 회사가 Pi를 일시 보유하면 필요할 수 있습니다. **전문 법률 검토 필수**.

### Q2: 개인 사업자로 시작할 수 있나?
**A**: 네. 초기 단계에는 개인 사업자(사업소득세)로 충분합니다. 향후 매출/직원 규모에 따라 법인 전환을 검토하세요.

### Q3: 해외 결제(Pi는 글로벌)에 대한 신고는?
**A**: Pi Network는 글로벌이지만, 회사의 주요 시장이 한국이면 한국 법령 적용. Supabase(싱가포르) 운영에 대해서는 개인정보 국외이전 신고만 필요.

### Q4: 청소년 회원 비율이 높으면 추가 신고가 필요한가?
**A**: 아니오. 청소년 보호정책 수립하고 필터링 운영하면 충분. 단, 학교 대량 가입 등 이상 패턴 시 모니터링 강화.

### Q5: 신고 후 변경 사항이 있으면?
**A**: 즉시 해당 기관에 변경 신고(보통 30일 내). 예: 서비스 기능 추가, 대표자 변경, 주소 변경 등.
