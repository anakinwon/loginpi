## 11. 지출 상한선 (Spend Cap)

Pro 요금제는 **지출 상한선이 기본 활성화** — 예상치 못한 대량 트래픽으로 요금이 폭등하는 것을 방지.

- 상한선 도달 시 해당 기능 일시 제한 (서비스 중단 아님)
- 사용량 80% 도달 시 이메일 알림
- 상한선 해제 가능 (Supabase 대시보드 → Billing)

---

## 12. 이 프로젝트에서 중요한 포인트

| 항목 | 현재 설정 | 비고 |
|---|---|---|
| **RLS** | 비활성화 | Service Role Key로만 접근 → 서버에서 모든 권한 제어 |
| **Service Role Key** | 서버 전용 (`SUPABASE_SERVICE_ROLE_KEY`) | 클라이언트 직접 사용 금지 |
| **단건 조회** | `.maybeSingle()` 사용 | `.single()`은 결과 없을 때 에러 → 금지 |
| **물리 DELETE** | 금지 | `del_yn = 'Y'` 논리삭제만 허용 |
