### 3. ⚠️ Cron Jobs (이 프로젝트 핵심 제약)

| 항목 | Hobby | Pro |
|---|---|---|
| **Cron 실행 주기** | **하루 1회 이하만 가능** | **임의 주기 (5분마다 등) 모두 가능** |
| 허용 예시 | `0 0 * * *` (매일 자정) | `*/5 * * * *`, `0 * * * *` 등 |
| 위반 시 | 배포 자체가 차단됨 (오류 로그도 없음) | 제한 없음 |

```
⚠️ 이 프로젝트 현재 설정: 0 0 * * * (매일 자정 1회)
   → /api/cron/event-reeval 이벤트 미션 재평가
   → 5분/10분/매시간 주기로 바꾸려면 Pro($20/월) 업그레이드 필요
   → 상세 이슈 기록: docs/TROUBLESHOOT.md
```
