## 작업 1: MAP 탭 LazySection 확대 적용

### 파일: `src/components/lbs/nearby-explorer.tsx`

**변경 사항:**
- 지도 뷰 + 등록 매장 목록을 `<LazySection>` 래퍼로 감싸기
- 근처 채팅방 목록도 `<LazySection>` 래퍼로 감싸기
- fallback: 차트 높이 스켈레톤 (높이 384px, 애니메이션)
- rootMargin: 50px (스크롤 진입 50px 전에 미리 로드)

**성능 효과:**
- 사용자가 MAP 탭에 진입했으나 아래로 스크롤하지 않으면 지도 UI 렌더 지연
- 대사용자 동시 접속 시 초기 JavaScript 파싱·평가 부하 감소
- Plotly 지도 라이브러리(무거움) 마운트 지연 → 초기 INP/FID 개선

## 작업 2: EVENT 탭 LazySection 확대 적용

### 파일: `src/components/event/client-event-gate.tsx`

**변경 사항:**
- 미션 목록 섹션 (`<div>` 아코디언)을 `<LazySection>` 래퍼로 감싸기

- M1~M10 미션 데이터 + 50명 랭킹 테이블 렌더 지연
- Core Web Vitals (LCP, CLS) 개선

| **표준 2: 비동기 검색** | 디바운스 300ms | ✓ 렌더 블로킹 없음, fetch 비동기 |

- 클라이언트 localStorage SWR(`readCache`/`writeCache`)과 조합

**기존 구현(유지):**
```typescript

useInfiniteScroll() // sentinel 감지
```
