# BEAN 토큰 발행 — 법무 자문 의뢰서

> **작성일**: 2026-06-17 · **의뢰 주체**: 아나킨(개인, Cafe.pi 운영자) · **관련 문서**: `docs/PRD_12_TOKEN.md` v1.7
> ⚠️ 본 문서는 외부 변호사·로펌 송부용입니다. 본 문서 자체는 법률 자문이 아니며, 아래 질의에 대한 **전문가의 서면 회신**으로 토큰 발행 방향을 최종 확정합니다.

## 1. 의뢰 배경 (1분 요약)

- **서비스**: Cafe.pi — Pi Network 메인넷 기반 온라인 커뮤니티 + O2O 마켓플레이스(채팅·마켓·음성채팅·이벤트). 현재 운영 중.
- **계획**: Pi Network의 공식 **Launchpad**를 통해 생태계 유틸리티 토큰 **BEAN**을 발행하려 함.
- **목적**: 자금 조달이 아닌 **사용자 보상·생태계 사용권**(기존 인앱 'Pi Bean' 팁의 온체인화). 사용자 획득·활성화 도구.
- **발행 주체**: **개인**(법인 아님) — 본 자문에서 개인 발행의 적정성·법인화 필요성도 함께 검토 요청.

## 2. 토큰 핵심 파라미터

| 항목 | 값 |
|---|---|
| 토큰명 | BEAN |
| 총 발행량 | 1,000,000,000개 (10억) |
| 세일가격 | 0.01 Pi / BEAN (`1 Pi = 100 BEAN`) |
| 초기 FDV | 10,000,000 Pi |
| 분배 | 세일 40% / 생태계 리저브 25% / 유동성 15% / 마케팅 12% / 팀 8% |
| 세일 구조 | Launchpad Fair-Access(약정 1~500 Pi). **참여자의 Pi는 발행사가 직접 수취하지 않고 유동성 풀에 예치** |
| 유통 | DEX 유동성 BEAN/Pi 페어 단독 (Pi 외 암호화폐·법정화폐 페어 미사용) |
| 발행 체인 | Pi Network(Stellar 기반) / Soroban 스마트 컨트랙트 |

## 3. 질의 항목

1. **증권성 판단**: BEAN(총 10억 개, 세일가 0.01 Pi)이 자본시장법상 '증권'(특히 투자계약증권)에 해당하는가? 해당 시 발행·청약 권유에 필요한 절차는?
2. **유틸리티 포지셔닝의 유효성**: "생태계 사용권·보상" 중심으로 설계·홍보할 경우 증권성 배제가 가능한가? 실질 판단에서 어떤 요소가 결정적인가?
3. **Launchpad 구조의 영향**: 세일 Pi가 발행사로 직접 귀속되지 않고 유동성 풀에 예치되는 구조가 증권성·자금조달 규제 판단에 유리하게 작용하는가?
4. **특금법(VASP)**: BEAN 발행·유통으로 가상자산사업자 신고 의무가 발생하는가? AML/KYC 요건은?
5. **세금**: 토큰 발행·세일·사용자 보상 지급의 부가세·소득세 취급은? (개인 발행 기준)
6. **표시·광고 규제**: 백서·랜딩 페이지에서 금지해야 할 표현, 필수 고지사항(리스크 경고 등)은?
7. **발행 주체 — 개인 vs 법인**: 개인 발행 시 증권성·특금법 책임이 개인에게 직접 귀속되어 위험이 크다고 판단됨. **법인 설립 후 발행이 더 안전한가?** 권고 구조는?
8. **역외 규제**: 글로벌 203개 언어 서비스 특성상 미국(SEC)·EU(MiCA) 등 역외 규제 노출 범위와 대응은?

## 4. 회신 요청 형식

- 각 항목별 **결론 + 근거 + 권고 조치**
- 특히 7번(개인/법인)과 1번(증권성)은 **발행 가부의 전제**이므로 우선 회신 요망.
